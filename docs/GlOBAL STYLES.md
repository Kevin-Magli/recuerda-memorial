- background use default gradient and border radius

Title
- TEXT SIZE 40px (or find a more flexible sizing)

- default highlights

Descriptions
- text size 16
- normal 400 weight
- default highlights

Button - Variation 1
- default gradient background
- bold text
- default radius
- padding 30 17
Button - Variation 2
- white background
- color 1 text and border (thin border but not pixelated)

Images
- Containers with image inside

Inner Sections

Container

Icon Card - MAIN

- background follows default gradient

Icon Card - Others
